# Expression Tree - C++

📌 **Project Overview**  
This C++ program builds a **binary expression tree** from a postfix expression and evaluates or prints it in infix form.

🚀 **Features**  
- Build tree from postfix  
- Print infix expression  
- Supports `+,-,*,/` operators  

🛠️ **How to Run**
```bash
g++ expression_tree.cpp -o expression_tree
./expression_tree
```

**Sample Output:**
Infix expression: ((a+b)*(c*(d+e)))
