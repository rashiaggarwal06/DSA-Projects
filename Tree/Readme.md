# Tree Mini Projects (C++)

This repository contains **5 mini Tree programs** implemented in C++. Each demonstrates different tree operations and algorithms.

## Programs and How to Run

1. **Binary Search Tree (BST)** (`bst.cpp`)  
```bash
g++ bst.cpp -o bst
./bst
```

2. **Tree Traversals** (`tree_traversals.cpp`)
```bash
g++ tree_traversals.cpp -o tree_traversals
./tree_traversals
```

3. **Height of Binary Tree** (`tree_height.cpp`)
```bash
g++ tree_height.cpp -o tree_height
./tree_height
```

4. **Lowest Common Ancestor (LCA)** (`lca.cpp`)
```bash
g++ lca.cpp -o lca
./lca
```

5. **Expression Tree** (`expression_tree.cpp`)
```bash
g++ expression_tree.cpp -o expression_tree
./expression_tree
```
