# Tree Traversals - C++

📌 **Project Overview**  
This C++ program demonstrates **inorder, preorder, postorder, and level-order traversals** on a binary tree.

🚀 **Features**  
- DFS: Inorder, Preorder, Postorder  
- BFS: Level-order traversal  

🛠️ **How to Run**
```bash
g++ tree_traversals.cpp -o tree_traversals
./tree_traversals
```

**Sample Output:**
Inorder: 4 2 5 1 3
Preorder: 1 2 4 5 3
Postorder: 4 5 2 3 1
Level Order: 1 2 3 4 5
