# Lowest Common Ancestor (LCA) - C++

📌 **Project Overview**  
This C++ program finds the **lowest common ancestor of two nodes** in a binary tree.

🚀 **Features**  
- Recursive solution for LCA  
- Works for any binary tree  

🛠️ **How to Run**
```bash
g++ lca.cpp -o lca
./lca
```

**Sample Output**
LCA of 6 and 2 is 5
