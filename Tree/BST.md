# Binary Search Tree (BST) - C++

📌 **Project Overview**  
This C++ program implements a **Binary Search Tree (BST)** with insert, search, and inorder traversal operations.

🚀 **Features**  
- Insert nodes into BST  
- Search for a value  
- Inorder traversal display  

🛠️ **How to Run**
```bash
g++ bst.cpp -o bst
./bst
```

**Sample Output:**
Inorder Traversal: 20 30 40 50 60 70 80 
60 found in BST
