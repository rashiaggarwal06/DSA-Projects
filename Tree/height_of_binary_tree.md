# Height of Binary Tree - C++

📌 **Project Overview**  
This C++ program calculates the **height of a binary tree** using recursion.

🚀 **Features**  
- Compute height from root to leaf  
- Can be extended to calculate diameter  

🛠️ **How to Run**
```bash
g++ tree_height.cpp -o tree_height
./tree_height
```

**Sample Output:**
Height of the tree: 3
