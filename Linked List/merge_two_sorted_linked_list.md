## 5️⃣ Merge Two Sorted Linked Lists
📌 **Project Overview**  
Merge two sorted singly linked lists into a single sorted linked list.  

🚀 **Features**  
- Recursively merges two sorted lists  
- Displays merged sorted list  

🛠️ **How It Works**  
1. Compare the first nodes of both lists.  
2. Recursively merge the remaining nodes.  
3. Return the head of the merged list.  
4. Display the merged list.  

**Sample Output**
1 -> 2 -> 3 -> 4 -> 5 -> 6 -> NULL
