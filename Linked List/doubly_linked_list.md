## 2️⃣ Doubly Linked List
📌 **Project Overview**  
This program implements a doubly linked list allowing traversal both forward and backward.  

🚀 **Features**  
- Insert elements at the end  
- Forward traversal  
- Backward traversal  

🛠️ **How It Works**  
1. Each Node contains `data`, `next`, and `prev`.  
2. Insert elements at the end and update `prev` pointer.  
3. Display forward by traversing `next`.  
4. Display backward by traversing `prev` from the last node.  

**Sample Output**
Forward: 10 -> 20 -> 30 -> NULL
Backward: 30 -> 20 -> 10 -> NULL
