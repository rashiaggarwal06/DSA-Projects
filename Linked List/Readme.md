# Linked List Mini Projects (C++)

This repository contains **5 mini Linked List programs** implemented in C++. Each demonstrates different linked list operations.

## Programs and How to Run

1. **Singly Linked List** (`singly_linked_list.cpp`)  
```bash
g++ singly_linked_list.cpp -o singly_linked_list
./singly_linked_list
```

2. **Doubly Linked List** (`doubly_linked_list.cpp`)
```bash
g++ doubly_linked_list.cpp -o doubly_linked_list
./doubly_linked_list
```

3. **Circular Linked List** (`circular_linked_list.cpp`)
```bash
g++ circular_linked_list.cpp -o circular_linked_list
./circular_linked_list
```

4. **Linked List Palindrome Checker** (`palindrome_linked_list.cpp`)
```bash
g++ palindrome_linked_list.cpp -o palindrome_linked_list
./palindrome_linked_list
```

6. **Merge Two Sorted Linked Lists** (`merge_sorted_lists.cpp`)
```bash
g++ merge_sorted_lists.cpp -o merge_sorted_lists
./merge_sorted_lists
```
