## 1️⃣ Singly Linked List
📌 **Project Overview**  
This C++ program implements a basic singly linked list with insert, delete, search, and display functionalities.  

🚀 **Features**  
- Insert elements at the end  
- Delete elements by value  
- Display all elements  

🛠️ **How It Works**  
1. Create a Node structure with `data` and `next` pointer.  
2. Insert elements by traversing to the end of the list.  
3. Delete an element by value by adjusting the `next` pointer of the previous node.  
4. Display all elements by traversing from head to NULL.  

**Sample Output**
10 -> 20 -> 30 -> NULL
10 -> 30 -> NULL
