## 4️⃣ Linked List Palindrome Checker
📌 **Project Overview**  
This program checks if a singly linked list is a palindrome (reads the same forwards and backwards).  

🚀 **Features**  
- Checks for palindrome using stack  
- Works for any size linked list  

🛠️ **How It Works**  
1. Traverse the list and push all elements onto a stack.  
2. Traverse the list again and compare each element with the top of the stack.  
3. If all match and stack is empty → Palindrome, else → Not Palindrome.  

**Sample Output**
1-> 2-> 1 = true
3-> 4-> 3-> 2 = false
